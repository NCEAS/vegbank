VegBank provides a suite of SOAP-based services that can be integrated easily into any Web or desktop application. Use of these services is advantageous to a software development effort as their use will mitigate the need for the large volumes of data needed to support spatial discovery of vegetation diversity and abundance, as well as the need for resources to perform analysis on these data. In essence, these services allow access to the thousands of vegetation plots stored in the VegBank system and the millions of places registered in the Vegbank gazetteer. This framework is deployed through Internet standards including HTTP, WSDL, UDDI, and XML. It uses XML-based Simple Object Access Protocol (SOAP) to communicate, making it compatible with the majority of Web service frameworks available today.
