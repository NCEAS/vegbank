Vegetation Plots Database Model 2000

References column:

When a field in a table refers to another, it is called a 'foreign key'; the field to which it refers is called its parent key. This column
contains the link to the parent key of the foreign key in this row. Below are flags which may also be added to the column.

E - Constraint is ENABLED.

D - Constraint is DISABLED.

N - Constraint has a NO ACTION delete rule.

C - Constraint has a CASCADE delete rule. If the parent value is deleted, the row in this table will be deleted as well.
