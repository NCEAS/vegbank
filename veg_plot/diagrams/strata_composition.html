Vegetation Plots Database - working draft 7/30/2000
 

strata_composition (possible change table name to species_vert_structure)

plot_id - System assigned unique identifier for each individual plot.  At this time a vegetation plot is defined by both discrete location (a single point) and date; meaning that two plots may exist for a single location if the supporting data were collected at differing times.  Also, sub-plots having smaller lateral coverage may exist within a larger regional plot.

species_original - name of a plant species as it was named in the plot collection

stratum_type - stratum level (moss, herb, bush, tree, etc.)

percent_coverage - this is the coverage of the named species within a stratum
