Vegetation Plots Database - working draft 7/30/2000
 

plot_master

plot_id - System assigned unique identifier for each individual plot.  At this time a vegetation plot is defined by both discrete location (a single point) and date; meaning that two plots may exist for a single location if the supporting data were collected at differing times.  Also, sub-plots having smaller lateral coverage may exist within a larger regional plot.

author_plot_number - this is the author's plot description number used in a publication to refer to this plot

previous_plot_record - if the plot represents a re-measurement provide the original plot_id, this is self-referential and it should be a foreign key to plot_id

date - day/month/year the plot was described

date_accuracy - plus/minus number of days

landOwner - owner of the land on which the plot was sampled

samplingMethod - method used for plot sampling (including, but not limited to: Daubenmire, modified Daubenmire, Zuric-Montpellier, North Carolina Vegetation Survey and TNC

effort_level - the effort spent describing the plot (very thorough, average effort, hurried)

vegUniformity - a measure of the small scale heterogeneity (see the Damman document)

element_cover - percent cover of element

class_association - name or code of association to which the plot belongs.  This will probably not be included because it is interpretive and thus does not belong here

class_alliance  - name or code of alliance to which the plot belongs.  This will probably not be included because it is interpretive and thus does not belong here

classification_quality - with what degree of confidence could the plot record be assigned to a recognized association.  Again, this is based on an interpretation
