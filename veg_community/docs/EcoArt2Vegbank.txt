INTRODUCTION:

The Ecoart communities data is stored in an MS Access mdb  file that is in the Ecoart distribution. 
In Ecoart version 2.65 this file was called "Ecoart data.mdb".

The Vegbank system is looking to populate all levels of the communities hierarchy  into the database. 
However each level is stored in a separate table in Ecoart, so specific mapping must occur for each level.

Levels:
The following levels exist "class",  "subclass", "group",  "subgroup",  "formation", "alliance", "association"  



DEFAULT values for all EcoArt communities and levels.

  ->  pubdate = "13-FEB-2002"

COMMCONCEPT
-------------------------
