VegBank XML

This page links you to example XML documents that contain VegBank data. These would be suitable for importing into the database. There are also links to schema files, which validate the XML documents which we receive. You may download any of these. Downloading and saving the schema to your local machine will make validation quicker for you. However, you may also link to a schema shown here to validate from our server.
