VegBank Help -- Download Formats

  • XML- This download format is the native VegBank XML. The entire VegBank database is capable of being contained in this XML document. Please see the information page for more information on our XML document.
  • ASCII- If you download data as flat-ASCII, you will get two documents. Each will be text, with field values separated by commas. This is a standard format that you can easily import into a database or spreadsheet program, as well as analysis programs such as PC-Ord. The first file will contain all plot-level data, and the second file will contain data about species abundance.
  • HTML- HTML downloads allow you to save the view of plots that is available after querying. The same can be accomplished (one plot at a time) by saving the HTML file that results when you view a plot.

Future download formats

We hope to allow users to download in the following formats in the near future:

  • VegBranch- This format will allow users to download plots and then import them into VegBranch, VegBank's desktop client.
  • PC-Ord- This format is similar to ASCII, but will be consistent with PC-Ord naming conventions.
  • TurboVeg- This download will be easily imported into TurboVeg, a popular database system.
