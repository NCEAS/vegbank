Version 2.0 of the ESA Standards released (March 31, 2003)

The ESA Vegetation Panel has completed Version 2.0 of the Guidelines for Describing Associations and Alliances of the U.S. National Vegetation Classification. The document is available as a pdf file [1.2M] and as a zipped doc file [199K].

The older version, Standards for Associations and Alliances of the US National Vegetation Classification version 1.0, is still available here, as a pdf file [1.5M] and as a zipped doc file [138K].

The Panel welcomes proposals for revisions. Proposals should be directed to the Chair of the Panel.
