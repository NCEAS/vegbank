VegBank XML

This page links you to example XML documents that contain VegBank data. These would be suitable for importing into the database (but please do not upload these as they are only samples). There are also links to schema files, which validate the XML documents which we receive. You may download any of these. Downloading and saving the schema to your local machine will make validation quicker if you validate your own XML documents locally. However, you may also link to a schema shown here to validate from our server.
