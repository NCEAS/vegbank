Help for 'Load Data'

This form takes data from a digital format (spreadsheet file, the generic .csv or comma-separated values; or Access database) and imports these data into one of the loading tables in VegBranch.

First select which table you'd like to import to, then find the file and import it. If field names do not match, you'll be directed to a new form where you can map your field names onto VegBranch loading table field names. If your file is a matrix file, it can be normalized using a button that links to a utility in vegbranch that normalizes this file. @vegbr-demo-linksfromform-master_switchboard@ @webpage_footer_html@
