Help for 'Confirmation Messages -- Options'

This form is shown when VegBranch starts if any confirmation messages are turned on in MS Access (machine-level setting). VegBranch will not function well with confirmation messages, because there will be too many of them. This form provides more information about how VegBranch should deal with confirmation messages. @vegbank_footer_html_tworow@
