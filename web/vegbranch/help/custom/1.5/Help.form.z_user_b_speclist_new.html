Help for 'Species List Loading Form (one species/taxon per screen)'

This form is for adding new plant concepts to the vegbank module. It is not for irregular taxa, like Carex sp. which can be handled without adding new plant concepts to the database. These new plant concepts should be valid taxonomic concepts that were not preloaded into VegBank. @vegbank_footer_html_tworow@
