Summary: Data about the soils collected from a plot.
Description: Soils data of standard types are listed in this table. For attributes that are not standard, userDefined fields must be used due to differences in methods of calculating soil variables. Each plot is linked to Plot Data by authorPlotCode.
Required in VegBranch: Not required.
Represents VegBank Tables: soilObs
VegBranch can generate for you: no
Questions: Were soils data collected on the plots?
Expertise Level: medium

Fields (Columns) in this table:

More information about fields available here.

» Required fields; you should always fill in values for these:
AuthorPlotCode Author's Plot number/code, or the original plot number if taken from literature. This "points" to the right plot in the Plot Data Loading Table.
soilHorizon The horizon to which this observation applies.
» Commonly used fields:
soilDepthTop The depth at which the horizon observation starts.
soilDepthBottom The depth at which the horizon observation ends.
soilColor Soil color (USDA guidelines recommended).
soilOrganic Percent organic content of the soil (for methods see observation.methodsNarrative).
soilTexture Soil texture class.
soilSand Percent sand in the soil hoizon.
soilSilt Percent silt in the soil horizon.
soilClay Percent clay in the soil horizon.
soilCoarse percent coarse fragments in the soil, prior to removal for textural analysis
soilPH pH of the soil (for methods see observation.methodsNarrative).
exchangeCapacity Cation exchange capacity (for methods see observation.methodsNarrative).
baseSaturation Percent base saturation (for methods see observation.methodsNarrative).
soilDescription Text description of the soil.
