VegBranch Loading Tables Overview -- Disturbance Data (G)

Navigation between loading tables:
Party Information | Species List | Plot Data | Plot Normalized | Presence Data | Strata Cover Data | Stem Data | Soil Data | Disturbance Data | Community Concept | Plot Communities | User Defined Metadata

Summary: Data about evidence of disturbance observed on the plot.
Description: Different types of disturbance can be described in this table. Plots may have many different types of disturbance, each with its own record in this table. Each plot is linked to Plot Data by authorPlotCode.
Required in VegBranch: Not required.
Represents VegBank Tables: disturbanceObs
VegBranch can generate for you: no
Questions: Was disturbance evidence recorded for the plots?
Expertise Level: medium

Fields (Columns) in this table:

More information about fields available here.

» Required fields; you should always fill in values for these:
authorPlotCode Author's Plot number/code, or the original plot number if taken from literature. This "points" to the right plot in the Plot Data Loading Table.
disturbanceType The type of disturbance being reported
» Commonly used fields:
disturbanceIntensity Intensity or degree of the disturbance.
disturbanceAge Estimated time in years since the disturbance event.
disturbanceExtent Percent of the plot that experienced the event.
disturbanceComment Text description of details of the disturbance and its impact on the vegetation.
» Internal VegBranch fields, please do not edit directly:
Z_disturbanceObsID Database generated identifier assigned to each unique disturbance observation.
