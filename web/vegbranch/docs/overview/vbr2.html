Once your plots are loaded, you can use the "Plot Query" to search for plots in VegBank. You should be able to find the plots you just added. You can also search for other plots, of course. After you find plots, you can download them into a "VegBranch Import" file (which is a .csv file), if you'd like to send these plots into VegBranch.
