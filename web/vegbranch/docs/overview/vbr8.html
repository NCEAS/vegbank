You may have some plots in data files on your computer that you'd like to add to VegBank.org. These are represented here as .csv (comma-separated values) and .mdb (Access database) files. The .csv files are spreadsheet files that contain values only (no formulas) and can be made from any spreadsheet program (Excel, 1-2-3, etc.). To start importing data, you need you data to be in one of these two file types. If you don't care which format you use, you should use Access databases, as the import will be faster with them.
