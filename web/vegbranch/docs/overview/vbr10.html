Here we introduce a second database, called the VegBank Module. This, too, is a Microsoft Access database that is on your computer's hard drive (shown in the yellow triangle). This database is a copy of the VegBank.org database and is used by VegBranch to store all data that conforms to VegBank.org's data format. The next slides will show you how this works.
