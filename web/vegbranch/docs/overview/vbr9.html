In order for VegBranch to use the VegBank Module, a connection or "link" between the two must be established. VegBranch can do this for you automatically (if the VegBank Module is in the same folder as VegBranch), or you can choose what VegBank Module you'd like to link to. The advantage of this approach is that you can have several different VegBank Modules, each with different data in it (or different versions of the same data). If you want to backup your files, you should make a copy of both VegBranch and the VegBank Module.
