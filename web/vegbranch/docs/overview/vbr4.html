Once your data have migrated into the VegBank Module, you can export this data into an XML file to send to VegBank.org. An XML file is a text file with instructions that VegBank.org can read to understand what the contents of your data are. XML files can get quite large, even for a limited set of plots. Zipping (compressing) your XML files after this step is recommended. VegBranch can help you do that, if you have WinZip on your computer.
