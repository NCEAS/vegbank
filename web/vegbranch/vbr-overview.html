VegBranch

Download | Overview | Loading Module | Help

VegBranch is a Microsoft Access database designed primarily to import legacy vegetation plot data, modify those data to conform to VegBank standards, migrate the data to the VegBank format, then prepare the data to be sent to the VegBank server (vegbank.org). There are help files associated with VegBranch that describe each function of VegBranch in detail. VegBranch now supports downloading of data from VegBank so that one can easily manipulate and analyze data on their home computer (see the main menu in VegBranch for details). The following overview helps show the relationship between VegBranch and the other VegBank components. Click the overview to see a larger version in a new browser window.

Explanation of Overview, Form Names in italics

VegBranch can Load Data from files on your computer, either .csv (comma separated values, a generic spreadsheet format) or .mdb (Access database), into the loading module of VegBranch. The Loading Module is a set of generic ecological tables in VegBranch, which simplify the vegbank datatable structure greatly, allowing easier import and data checking.

Once your data are in the Loading Tables, you then provide VegBranch with some Metadata (which is often missing in data files),
Translate Data from your format into the VegBank data format,
Data Check for errors,
then Migrate Data to the Vegbank Module.

The VegBank Module is a connected MS Access database which mimics vegbank.org's database structure exactly. Once migration has been completed and your data are in the vegbank module, you have a small version of the vegbank database on your computer.

To send this data to vegbank.org for inclusion in the central database there, you will need to Export Data to an XML file which is a specially formatted text file that can hold vegbank data. This file can be uploaded to vegbank.org, and the data in it will be used to add plots, parties, project, plants, communities and the rest of your data to the vegbank.org database.

Note that the yellow triangle shows what parts of the process take place on your computer, and the green one shows what happens on the internet. VegBranch operates completely on your computer. The only interaction with vegbank.org is with the XML files.

NEW! You can now download data from vegbank.org to import into VegBranch. Download a "VegBranch Import" file from vegbank.org. This is a specially formatted .csv file based on the VegBank XML file. Just as with the loading process, the interaction between VegBank and VegBranch is entirely via uploaded and downloaded files. See the main menu to link to the form to import data. Documentation of this functionality is available from that form.
