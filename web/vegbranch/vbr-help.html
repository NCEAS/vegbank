Download | Overview | Loading Module | Help

This page describes VegBranch's help section. Almost every button, field, picklist, panel, etc. is described locally in VegBranch. You should right-click on something in VegBranch that you don't understand, and select "Help" from the menu of options there. This takes you to a description of the form and the item you right-clicked on. From that screen, there is also a link to online help that is in not completely finished.

Email us for help

If you are still having trouble, or help hasn't been finished for a form that you would like help with, please email us at help@vegbank.org.

Also, you may find the overview of the loading tables helpful.
