Cite or Link to VegBank

Why Cite or Link?

One of the biggest advantages of VegBank is that the data that are stored in it are permanently archived. This means that you can cite a plot in VegBank, and others will be able to follow your citation hyperlink to see the same data you saw.

How do I cite?

Citation to VegBank requires that you know the Accession Code of the plot (or other data) to which you want to link. An Accession Code is a unique code that tells VegBank what data you are interested in. This can be found on the detailed views of plots (and other data).

Once you know the Accession Code of a plot, you can cite it by using the following URL (to type into a web browser) formula:

http://vegbank.org/cite/[AccessionCode]

[AccessionCode] is the Accession Code of the plot you are interested in.

Example

Cite this plot with the following URL: http://vegbank.org/VB.Ob.3736.GRSM125

Cite this community with the following URL: http://vegbank.org/VB.CC.5728.CEGL003906

Cite this plant with the following URL: http://vegbank.org/VB.PC.54588.SPOROBOLUSAIROI

You can cite anything in VegBank that has an Accession Code like this.
