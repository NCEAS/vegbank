VegBranch

VegBranch (alpha 1.0) [This is a 46 Mb zip file that expands to 157 Mb]
VegBranch is a Microsoft Access database designed primarily to import legacy vegetation plot data, modify those data to conform to VegBank standards, migrate the data to the VegBank format, then prepare the data to be sent to the VegBank server (www.vegbank.org). There are help files associated with VegBranch that describe each function of VegBranch in detail. VegBranch will eventually support downloading of data from VegBank so that one can easily manipulate and analyze data on their home computer.
