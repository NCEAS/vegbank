VegBank is being upgraded to a new server

How will this affect me?

VegBank will be read-only for a short time, about 30 minutes. There will be a message at the top of each screen indicating VegBank is read-only.

While VegBank is read-only, you will be able to:

  • Browse the website normally.
  • Search for plots, plants, communities.
  • Add items to a dataset -- however these items will be lost when the new VegBank goes online.
  • Download datasets, map data.

While VegBank is read-only, you will not be able to:

  • Add plots.
  • Interpret plots to a community.
  • Interpret plants.
  • Register as a user of VegBank
  • Request certification

Thank you for your patience. We hope that the short inconvenience will be worth the faster, more efficient VegBank server that will soon be ready.
