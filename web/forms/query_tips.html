  TIPS FOR DEVELOPING QUERIES
  • Query Criteria
Plot Code -- name that the author applied to the plot
State -- state in which the plot was collected
Elevation -- elevation at which the plot was collected
Plant Taxon -- a string of plant names
    (comma separated with no spaces)
    potentially included in the plot observations
Geology -- the surficial geology at the plot collection location
Plot shape -- the shape of the plot
Slope gradient -- slope gradient in degrees
Slope position -- slope postion includes:
Slope aspect slope aspect in degrees
Latitude -- latitude of the origin of the plot
Longitude -- longitde of the origin of the plot
  • Query Operators
contains -- plot that has one of the following
is equal to -- criteria is equal to the value
contains -- plot that has at least one of the following values
is greater than -- criteria is greater than the value
    (for numeric values)
is less than -- criteria is less than the value
begins with -- critiera starts with the string value
ends with -- criteria ends with the string value
  • Query Criteria Values
operator: is equal to -- single token (string or numeric value)
operator: contains -- comma separated tokens (with no spaces)
operator: is greater than -- single numeric value
operator: is less than -- single numeric value
operator begins with -- single string value
operator ends with -- single string value
  • Query Examples
'State is equal to WY'

'Plant Taxon contains 1,2,3,4'
