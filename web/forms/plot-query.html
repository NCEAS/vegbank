Plot Query Form
This form can be used to construct and issue simple queries against
the VegBank plots database. Currently there are ~1200 plots stored in
the archive. These plots were collected by the Nature Conservancy
from the following National Parks:

Zion National Park -- Utah
 
Query Type:
Please choose the type of query based on the number of attributes you would like to use for the database selection.
single attribute query -- input an attribute in a single attribute below
multiple attribute query -- input a value for each attribute below

choose to enter multiple use commas to separate the plant names.
